# Whatsapp SDK

Connection SDK Whatsapp API ESI

## Installation

This project using composer.

```
$ composer require esistemasinteligentes/whatsapp-sdk
```

## Uso

```php
<?php

use ESI\WhatsappSDK;

$whatsapp = new WhatsappSdk('instance', 'token');
```
